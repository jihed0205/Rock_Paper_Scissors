# 🎮 Rock Paper Scissors

A simple **Rock Paper Scissors** game built with **HTML, CSS, and JavaScript**.

## Features

* Play against the computer
* Random computer moves
* Displays:

  * Your move
  * Computer's move
  * Game result (Win, Lose, Tie)
* Score tracking:

  * Wins
  * Ties
  * Losses
* Reset score button
* Interactive animations and hover effects

## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
